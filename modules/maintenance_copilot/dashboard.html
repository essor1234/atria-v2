What this does

Retrieve, validate, and cross-reference maintenance documentation (AMM, MEL, CDL, TSM) grounded in the approved manuals. Every recommendation is cited; every graph link starts unverified until an engineer confirms it. Dispatch is never automated.

Guardrails

  • Mandatory citation — uncited sentences are dropped from synthesized answers.
  • Confidence threshold — low-confidence results are routed for manual review.
  • Version-aware retrieval — defaults to the current approved revision.
  • Audit trail — every query, recommendation, and confirmation is logged.

CLI (also available in chat)

copilot.py query · recommend-refs · validate · check · graph build/show/confirm · audit
